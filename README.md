# lumi

Сообщество проверенных анкет для знакомств. Демо-приложение на **Next.js 14 + Tailwind CSS** с анимациями (Framer Motion), тёмной темой и мобильной адаптацией с нижней навигацией.

> Это легальный продукт-каталог знакомств: проверенные профили, интересы, чат и встречи. Без интим-услуг и почасовых тарифов.

## Что внутри

- 🏠 **Главная** — сторис-баннеры, фильтр-категории, сетка анкет
- 👤 **Анкета** — галерея фото, параметры, интересы, карта, отзывы, кнопка «Написать»
- 🗂 **Каталог**, ❤️ **Избранное**, 🗺 **Карта**
- 📱 Мобильная нижняя навигация (Главная / Каталог / Избранное / Карта / Профиль)
- ✨ Плавные анимации, тёмная тема, бейджи верификации

## Стек

- Next.js 14 (App Router) + TypeScript
- Tailwind CSS 3
- Framer Motion, lucide-react
- Фото — заглушки с picsum.photos, карта — OpenStreetMap embed

## Запуск

```bash
npm install      # или: bun install
npm run dev      # http://localhost:3000
```

Сборка: `npm run build && npm run start`

## Структура

```
app/
  page.tsx              # Главная
  catalog/ favorites/ map/
  profile/[id]/page.tsx # Страница анкеты
  components/           # Header, BottomNav, StoryRail, ProfileCard, Gallery…
  data/profiles.ts      # Моковые данные анкет
```
